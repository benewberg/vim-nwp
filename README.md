* https://nofilmschool.com/2017/05/watch-colors-twin-peaks

## Installation
### Using vim-plug
Add this to your config:
`Plug 'benewberg/vim-nwp'`

Then,
`:PlugInstall`

Lastly, in your config:
`colorscheme nwp`
